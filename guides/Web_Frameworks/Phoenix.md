# Phoenix
[Phoenix](http://phoenixframework.org/) is a framework written in Elixir which implements the server-side MVC pattern. It is built for web development.It is a good option today in the market.The website of the phoenix says **A productive web framework that does not compromise speed and maintainability**.This is true in fact because earlier developers had a myth either speed or quality both can not go hand in hand one need to compromise the either but phoenix breaks the myth and therefore it carved its name among the most popular and fastest web frameworks.Phoenix is a modern web framework that helps building APIs and web applications easy.
